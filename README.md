# OxiEngine
Written in C++, OxiEngine's goal is to create unforgetable horror experiences.
Inspired by Frictional Games and their HPL Engine. 
## Uses:
- SDL 2  
- GLEW  
- glm  
## Features:
SDL OpenGL-enabled Window initialization
## Planned Features:

Perspective 26-10-2018
